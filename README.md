# E-COMMERCE-FULL-FUNCTIONAL
TO MAKE A FULLY FUNCTIONAL E-COMMERCE


**for frontend**\
use node version 20.19.6 by using nvm(recommended)\
first install nvm in your system then install specific node version

**for backend**\
apt install python3.12-venv \
venv is a virtual environment in the backend folder\
use requirements.txt to mention your packages \
use pip freeze > requirements.txt to update when ever needed


**for starting backend**\
 fastapi dev app.py


**for starting frontend**\
npm run dev
